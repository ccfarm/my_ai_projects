##  smart team让 AI 团队帮你从一个 idea 生成可预览的网站。
- https://github.com/ccfarm/smart_team
- smart_team 是一个面向个人开发者、创业者和小团队的 AI 网站生成工具。你只需用自然语言描述想做的网站，AI 产品经理会先和你对话把需求聊清楚，确认后由 AI 开发者直接生成可运行的网站代码，并在页面内实时预览。生成后还能继续用对话微调。
