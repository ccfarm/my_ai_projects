### A 股投研助手
> [https://github.com/ccfarm/smart_team](https://github.com/ccfarm/simple_agent)
- 一个面向个人 A 股投资者的投研助手：维护关注股票列表，自动拉取指定股票的研报 / 新闻 / 公告，并基于这些本地资料用 AI（DeepSeek）进行 RAG 问答分析。多用户隔离，可本机运行，也可部署到一台云主机随时访问。

### AI项目开发团队
> [https://github.com/ccfarm/smart_team](https://github.com/ccfarm/smart_team)
- smart_team 是一个面向个人开发者、创业者和小团队的 AI 网站生成工具。你只需用自然语言描述想做的网站，AI 产品经理会先和你对话把需求聊清楚，确认后由 AI 开发者直接生成可运行的网站代码，并在页面内实时预览。生成后还能继续用对话微调。
- 目前的实现是一次性生成整个网站，所以只适合小项目
